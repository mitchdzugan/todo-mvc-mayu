# todo-mvc

1. `yarn`
2. `yarn watch`
3. `yarn scss` in new tab
4. `yarn start` in new tab

# Testing
1. `yarn test`
2. open [http://localhost:8021](http://localhost:8021)

# Production Build
1. `yarn build`
2. `yarn start`
